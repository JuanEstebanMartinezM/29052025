En esta tabla utilizamos la estructura base para la creacion de una tabla con etiquetas como:
table: etiqueta que define tabla
caption: etiqueta que asigna nombre o descripcion a la tabla, #se recomienda usarla como primer hijo de table
thead: etiqueta que indica que es el encabezado de la tabla
tr: etiqueta que crea las columnas de la tabla
th: etiqueta que crea celdas de encabezado
tbody: etiqueta que define el cuerpo de la tabla
td: etiqueta que cres celdas par datos simples
